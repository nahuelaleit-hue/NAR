# NAR - CMA Pro
**Análisis Comparativo de Mercado con Inteligencia Artificial.**

Esta aplicación permite realizar tasaciones inmobiliarias precisas comparando propiedades sujeto con comparables del mercado, utilizando el modelo **Claude 3.5 Sonnet** para el procesamiento de datos.

### 🚀 Funcionalidades
- **Login de Seguridad:** Acceso restringido mediante código.
- **Extracción con IA:** Pegá texto libre de avisos inmobiliarios y la IA extraerá los datos (m2, ambientes, precio) automáticamente.
- **Reportes Profesionales:** Generación de secciones de Ajustes, Valor Estimado y Conclusión.
- **Interfaz Premium:** Diseño oscuro (Dark Mode) optimizado para una experiencia profesional.

### 🛠️ Tecnologías
- **React.js**
- **Anthropic API (Claude)**
- **CSS-in-JS** para un estilo minimalista y funcional.
